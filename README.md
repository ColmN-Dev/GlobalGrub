# GlobalGrub

## Overview

GlobalGrub is a Flask web application built to allow users to search and browse recipes. It includes user authentication, a profile system, and the ability to save favourites. The focus of the project is on combining a simple backend with a polished frontend UI.

---

## Features

- Search functionality using `TheMealDB` API 
- Filter results by category, region, and ingredients  
- User authentication (sign up, login, logout)  
- Save favourite items using `localStorage`  
- Profile page with avatar selection  

---

## Tech Stack

- Backend: `Flask (Python)`  
- Frontend: `HTML`, `CSS`, `JavaScript`  
- Data Storage: `users.json`  
- Deployment: `Render.com`  

---

## Authentication

Authentication is handled using `Flask Sessions`.  
User data is stored in a `users.json` file with hashed passwords.
